# RORI Community FC - Official Website ⚽

Welcome to the official web repository for **RORI Community FC**. This is a single-page frontend application designed to manage club news, showcase the team roster, and handle new player registrations.

## 🌟 Features
* **Modern UI:** A clean, dark-themed interface optimized for sports clubs.
* **Responsive Design:** Fully fluid CSS Grid layout that adapts to mobile and desktop screens.
* **Player Registration:** A built-in HTML/JS form with validation for new community players to sign up.
* **Sticky Navigation:** Easy access to different sections.

## 🛠️ Built With
* **HTML5** - Page structure and form validation
* **CSS3** - Styling and CSS Grid
* **Vanilla JavaScript** - Frontend form submission handling
